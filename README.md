Our project compares different search algorithms to compare performance when solving an 8-puzzle problem.
The 3 algorithms compared were Breadth-First Search, Depth-First Search and A* search. Each was tested across 3 seperate test cases based on the number of inversions.
An 'easy' testcase with few inversions. a 'medium' testcase with several inversions and an unsolvable testcase with an odd number of inversions.

To run this, install libraries 
```
pip install -r requirements.txt
```
and then run main.py
```
python3 main.py
```
